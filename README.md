# Python Calculator

## 📌 Project Description

This is a simple **Python Calculator** program that performs basic arithmetic operations.
The program takes an operator and two numbers from the user and then performs the selected calculation.

The calculator supports the following operations:

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)

The program uses **Python conditional statements (if, elif, else)** to determine which operation to perform.

---

## ⚙️ Technologies Used

* Python 3
* Basic Python concepts (Input, If-Else statements, Operators)

---

## 🚀 How It Works

1. The user enters an operator (+, -, *, /).
2. The user enters two numbers.
3. The program checks the operator using **if-elif conditions**.
4. The result of the calculation is displayed.

---

## 💻 Example Output

```
Enter the operator (+ - * /): +
Enter the first number: 10
Enter the second number: 5
The result is: 15
```
